# Exercise Swap Guide

**Date:** 2026-03-02
**Purpose:** When equipment is taken at a crowded gym, swap to an equivalent exercise that hits the same muscle group in a similar movement pattern.

## Swap Rules

1. **Same muscle group** — obviously
2. **Same movement type** — if it's stretch-focused (fly, overhead extension), swap for another stretch exercise, not a shortened-position one
3. **Similar load profile** — don't swap a heavy compound for a light isolation unless you adjust volume
4. **Rest periods carry over** — use the same rest you would for the original exercise

## Push Day Swaps

### Chest
| Primary | Swap Options | Notes |
|---|---|---|
| Barbell Bench Press | DB Bench, Chest Press Machine, Smith Bench | Machine if no spotter available |
| Incline DB Press | Incline Barbell, Incline Machine, Low-to-High Cable Fly | Any incline pressing movement |
| Cable Chest Fly | DB Fly, Pec Deck Machine, Band Fly | All stretch-focused for chest |

### Shoulders
| Primary | Swap Options | Notes |
|---|---|---|
| DB Shoulder Press | Barbell OHP, Machine Press, Landmine Press | Any vertical pressing |
| Lateral Raise (DB) | Cable Lateral Raise, Machine Lateral Raise, Band Lateral | Cable = constant tension, better if available |
| Cable Lateral Raise | DB Lateral Raise, Machine Lateral Raise | Less constant tension but same target |

### Triceps
| Primary | Swap Options | Notes |
|---|---|---|
| Cable Tricep Pushdown | Dip Machine, Close-Grip Bench, Band Pushdown | Hits lateral/medial heads |
| Incline Cable Overhead Extension | DB Overhead Extension, Skull Crushers, Cable Overhead (standing) | All stretch long head — prioritize these |

## Pull Day Swaps

### Back
| Primary | Swap Options | Notes |
|---|---|---|
| Lat Pulldown | Pull-ups, Cable Pullover, Machine Pulldown | Pull-ups = bodyweight version |
| Seated Cable Row | DB Row, Chest-Supported Row, Machine Row, T-Bar Row | Any horizontal pull |
| Face Pull | Reverse Fly (DB or Machine), Band Pull-Apart | All rear delt focused |

### Biceps
| Primary | Swap Options | Notes |
|---|---|---|
| Cable Bicep Curl | DB Curl, Barbell Curl, Machine Curl | All supinated curls |
| Hammer Curl (DB) | Cable Hammer Curl (rope), Cross-Body Hammer | Brachialis focused |
| Reverse Fly | Face Pull, Band Pull-Apart, Rear Delt Machine | All rear delt |

## Leg Day Swaps

### Quads
| Primary | Swap Options | Notes |
|---|---|---|
| Barbell Squat | Hack Squat, Leg Press, Smith Squat, Goblet Squat | Hack squat closest to squat pattern |
| Leg Press | Hack Squat, Smith Squat, Barbell Squat | All heavy quad compounds |
| DB Lunge | Bulgarian Split Squat, Walking Lunge, Step-Up | All unilateral quad work |

### Hamstrings / Posterior Chain
| Primary | Swap Options | Notes |
|---|---|---|
| Barbell RDL | DB RDL, Cable Pull-Through, Good Morning | DB RDL = no rack needed |
| Seated Leg Curl | Lying Leg Curl, Nordic Curl, Swiss Ball Curl | Any knee flexion movement |

### Calves
| Primary | Swap Options | Notes |
|---|---|---|
| Leg Press Calf Raise | Standing Calf Raise, Seated Calf Raise, Smith Calf Raise | Standing = gastrocnemius, Seated = soleus |

## KB Day Note

KB days are conditioning-focused — if a specific KB isn't available, just use a different weight. The movement matters more than the exact load on easy days.

## Using This Guide at the Gym

When a station is taken:
1. Check this table for your current exercise
2. Pick the first available swap option
3. Use the same rep range and rest period
4. Log the swap in your session notes (so we know what you actually did)

The goal is never to skip a muscle group just because one piece of equipment is busy.
