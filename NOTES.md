## W3 Planning Notes

- **Morning flows need progression:** W1/W2 used the same flows. W3 should be harder and more varied. Ideas:
  - Add plank variations, bird dogs, or dead bugs to core work
  - Add tempo bodyweight squats or pistol progressions
  - Pull from existing floor exercises in the library (yoga flows, animal movements)
  - Push day flow: add push-up variations (diamond, archer, decline)
  - Pull day flow: add active hangs (L-hang, typewriter hang) instead of just dead hangs
  - KB day flow: integrate light KB work (halos, windmills) into the morning session
- **KB day philosophy:** Tuesday = "Explore" (introduce new/weird movements + traditional KB strength). Thursday = "Refine & Drill" (practice Tuesday's new stuff + conditioning). Keeps it fun while building real skill.
- **Cool KB movements to rotate in:** TGU, windmill, bent press, overhead lunge, Simba lift (squat front raise), lunge-snatch, cossack squat, dead clean, figure-8 variations. Library has tons of these.
- **Overall W2 feel:** Easier than W1 (body adapting). W3 should bump volume or intensity slightly.
