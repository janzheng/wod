**Date:** 2026-03-10
**Source:** [@DeanTTraining](https://x.com/deanttraining/status/2031013274335019251)

## The List

Dean's "best 1-3 exercises per muscle group" — heavily machine-biased for hypertrophy:

| Muscle Group | S Tier Picks |
|---|---|
| Chest | Seated/Flat Chest Press, Incline Chest Press, Pec Fly Machine |
| Lats | Wide Grip Lat Pulldown, Single Arm Lat Pulldown, Wide Grip Pull-Ups |
| Mid/Upper Back | Chest Supported T-Bar Row, Chest Supported Machine Row, Chest Supported Cable Row |
| Shoulders | Seated Machine OHP, Cuffed Cable Lateral Raises, Cable Rear Delt Flys |
| Biceps | Supinated Machine Curls w/ Arm Support, Preacher Cable Curls, Seated BTB Cable Curls |
| Triceps | Dip Machine, Incline Cable Tricep Pushdowns, Overhead Machine Extensions |
| Traps | Machine Kelso Shrugs, Smith Vertical Shrugs |
| Quads | Pendulum Squats, Leg Extensions, Leg Press (feet low) |
| Hamstrings | Seated Leg Curl, 45° Extension, Machine SLDLs |
| Glutes | Hip Thrust Machine, 45° Extension, Machine RDLs |
| Calves | Calf Press on Leg Press, Donkey Calf Raises, Standing Calf Raises |
| Abs | Ab Crunch Machine, Weighted Decline Sit-Ups, Cable Crunches |

## What We Agree With

- Machine-based training is great for hypertrophy, especially training solo without a spotter
- Chest-supported rows, lat pulldowns, leg press/extension/curl are all staples already in our program
- Cable work for shoulders and arms is effective and safe

## Our Training Philosophy

- **Dumbbells + KBs + machines** — no barbell dependence. Barbells feel unsafe without a spotter (bench, squat especially). DBs/KBs/machines give 90% of the stimulus with much less injury risk solo.
- **Functional + athletic focus** — not just hypertrophy. Want to be strong for daily use and movement, not just look big. KB work and morning flows support this.
- **Use what Arrillaga has** — no point chasing specialty equipment (pendulum squat, hip thrust machine, Kelso shrug machine, etc.)

## Available at Arrillaga (overlaps with list)

- Chest press / incline press machines
- Lat pulldown, cable row, chest-supported machine row
- Leg press, leg extension, seated leg curl
- Cable station (flys, lateral raises, curls, crunches)
- Dip station

## Not Available / Not Needed

- Pendulum squat, hip thrust machine, 45° extension machine
- Machine Kelso shrugs, machine oblique crunches
- Barbell compounds (by choice — no spotter)
