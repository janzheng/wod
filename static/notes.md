# Gym Quick Reference

---

## Rest Periods

| Exercise Type | Rest | Why |
|---|---|---|
| **Heavy compounds** (squat, leg press, rows, bench) | **3 min** | 90% recovery — don't rush these |
| **Moderate isolation** (curls, lateral raises, leg curls) | **2 min** | 80% recovery |
| **Light / pump** (cable flies, face pulls, band work) | **60-90 sec** | Pump > max output |
| **KB conditioning days** | **30-60 sec** | Metabolic training — short rest is the point |

**1 min rest = 30% effective. 2 min = 80%. Don't rush.**

## Exercise Swaps

If a machine is taken, swap to something that hits the same muscle.

### Push Day

| If taken... | Use instead |
|---|---|
| Barbell Bench | DB Bench, Chest Press Machine, Smith Bench |
| Incline DB Press | Incline Barbell, Incline Machine, Low-to-High Cable Fly |
| Cable Chest Fly | DB Fly, Pec Deck Machine |
| DB Shoulder Press | Barbell OHP, Machine Press, Landmine Press |
| Lateral Raise (DB) | Cable Lateral Raise, Machine Lateral Raise |
| Cable Tricep Pushdown | Dip Machine, Close-Grip Bench |
| Incline Cable Overhead Ext | DB Overhead Extension, Skull Crushers, Standing Cable Overhead |

### Pull Day

| If taken... | Use instead |
|---|---|
| Lat Pulldown | Pull-ups, Machine Pulldown, Cable Pullover |
| Seated Cable Row | DB Row, Chest-Supported Row, Machine Row, T-Bar Row |
| Face Pull | Reverse Fly (DB or Machine), Band Pull-Apart |
| Cable Bicep Curl | DB Curl, Barbell Curl, Machine Curl |
| Hammer Curl | Cable Hammer Curl (rope), Cross-Body Hammer |

### Leg Day

| If taken... | Use instead |
|---|---|
| Barbell Squat | Hack Squat, Leg Press, Smith Squat, Goblet Squat |
| Leg Press | Hack Squat, Smith Squat, Barbell Squat |
| Barbell RDL | DB RDL, Cable Pull-Through, Good Morning |
| Seated Leg Curl | Lying Leg Curl, Nordic Curl |
| DB Lunge | Bulgarian Split Squat, Walking Lunge, Step-Up |
| Leg Press Calf Raise | Standing Calf Raise, Seated Calf Raise |

## New Exercise: Incline Cable Overhead Tricep Extension

**Setup:** Bench at ~45° incline, facing away from cable. Rope on low/mid pulley behind you.

**Why:** Hits the **long head** (biggest part of triceps) in a deep stretch. Cable = constant tension. Bench = no cheating.

**Reps:** 3x10-12. Pair with pushdowns for full tricep coverage.

**If setup doesn't work:** Standing cable overhead extension or DB overhead extension.
